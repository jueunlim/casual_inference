4.1선형회귀의 필요성
  • 회귀분석은 100년이 넘는 역사를 가진 고전 모형이지만, 인과추론에서 여전히 매력적이고 강력하며, 때로는 위험한 도구입니다.

4.1.1모델이 필요한 이유
  • 은행 예시: 신용카드 한도(처치)가 채무불이행률(결과)에 미치는 영향을 추정하고 싶다.
  • 실제 데이터에서는 신용 한도와 채무불이행률 사이에 음의 상관관계가 나타남.
  • 이유: 은행이 위험이 낮은 고객에게 더 높은 한도를 설정하는 처치 배정 메커니즘 때문.
  • 이것은 교란편향(confounding bias)의 전형적 사례입니다.

차원의 저주와 선형회귀의 해결
  • 소득이나 신용점수 같은 대리변수(proxy)로 고유 위험을 추정할 수 있다.
  • 조건부 독립성 가정 (Y₀, Y₁) ⊥ T | X 이 만족되면 인과효과 식별 가능.
ATE = E_x { E[Y | T=1, X=x] − E[Y | T=0, X=x] }
  • 하지만 공변량 X가 많으면 각 셀의 표본이 부족해지는 차원의 저주curse of dimensionality 문제 발생.
  • 선형회귀는 이를 해결하는 차원 축소 알고리즘으로 볼 수 있다.
인과추론 용어 ↔ 머신러닝: 특성=공변량, 가중치=매개변수, 목표=결과

4.2회귀분석 이론 — OLS
  • 선형회귀분석은 평균제곱오차(MSE)를 최소화하는 매개변수 β*를 찾는다.
β* = argminβ E[ (Y_i − X_i β)² ]
  • 이를 미분하고 0으로 두면 정규방정식의 해를 얻는다:
β* = E[X'X]−1 E[X'Y]
β̂ = (X'X)−1 X'Y   (표본 버전)
  • X'X는 공분산/분산 역할, X'Y는 분자 역할 → 공분산을 분산으로 나누는 구조.

4.2.1단순선형회귀
  • 단일 설명변수 T만 있을 때, 매개변수는 다음과 같이 단순화:
τ̂ = Cov(Y, T) / Var(T) = E[(T_i − T̄)(Y_i − Ȳ)] / E[(T_i − T̄)²]
  • T가 무작위로 배정된 경우 τ̂는 ATE입니다.
  • 회귀분석은 처치와 결과의 공분산을 처치의 분산으로 나눈 값.
  • 직관: T가 1단위 변할 때 Y가 얼마나 함께 움직이는가를 정규화한 것.

4.2.2다중선형회귀
y_i = β₀ + τT_i + β₁X₁i + … + βₖXₖi + u_i
  • 설명변수가 두 개 이상일 때, 관심 있는 건 T와 관련된 τ 뿐!
  • τ̂는 다음과 같이 구할 수 있다:
τ̂ = Cov(Y, T̃) / Var(T̃)
T̃ = T를 모든 공변량 X에 대해 회귀한 잔차(residual)
  • 다중회귀에서 회귀계수의 의미: 모델의 다른 변수들의 효과를 고려한 후 얻은 동일 설명변수의 이변량 계수.
  • 인과추론 관점: 다른 변수를 사용해 T를 예측한 후 얻은 T의 이변량 계수.

4.3프리슈–워–로벨 정리와 직교화
  • FWL 스타일의 직교화orthogonalization는 가장 먼저 사용할 수 있는 편향 제거(debiasing) 기법.
  • 비실험 데이터를 처치가 무작위 배정된 것처럼 보이게 만듦.
  • FWL 정리에 따르면, 다중선형회귀분석을 세 단계로 나눌 수 있다:
1. 편향 제거 단계: T를 X에 대해 회귀 → 잔차 T̃ = T − T̂
2. 잡음 제거 단계: Y를 X에 대해 회귀 → 잔차 Ỹ = Y − Ŷ
3. 결과 모델 단계: Ỹ를 T̃에 대해 회귀 → 인과효과 추정
  • 이 세 단계의 결과는 한 번에 다중회귀한 것과 정확히 동일!

4.3.1편향 제거 단계
  • 교란 요인 X로부터 처치 T를 예측하는 회귀 모델을 적합.
  • 예측값을 빼서 잔차를 구함:
linẽ_i = line_i − linê_i + linē
  • 잔차 linẽ는 X의 다른 변수와 상관관계가 없는 버전의 처치.
  • 즉, 교란 요인의 영향이 제거된 "깨끗한" 처치 변수를 얻는다.
  • 시각화: 편향 제거 전에는 음의 추세 → 편향 제거 후에는 하향선이 사라짐.

회귀계수의 또 다른 공식
  • 단순선형회귀에서:
β₁ = E[(T_i − T̄) · y_i] / E[(T_i − T̄)²]
  • 다중선형회귀에서는:
β₁ = E[(T_i − E[T|X]) · y_i] / E[Var(T|X)]
  • 편향 제거만 적용해도 모든 교란 요인을 포함한 완전 모델과 동일한 편향되지 않은 추정값을 얻는다.
  • 시각화: 편향 제거된 신용 한도(x축) vs 채무불이행률(y축) → 더 이상 하향선이 없음!

4.3.2잡음 제거 단계 & 4.3.3표준오차
  • 잡음 제거: 결과 Y를 처치가 아닌 공변량 X에 대해 회귀하여 잔차를 얻는다.
default̃_i = default_i − default̂_i + default̄
  • 잡음 제거는 처치효과 추정값을 바꾸지 않지만, 분산을 줄여 표준오차를 낮춘다.
  • X가 Y의 분산을 많이 설명할수록 잡음 제거 효과가 크다.
표준오차 공식:
SE(β̂) = σ(ê) / [ σ(T̃) · √(n − DF) ]

ê = 회귀 모델의 잔차, n−DF = 자유도(매개변수 개수 차감)

잡음 제거의 핵심과 실험 설계
  • 결과를 잘 예측할수록 잔차가 작아지고 → 추정값의 분산이 낮아진다.
  • 처치가 결과를 많이 설명하면 매개변수 추정값의 표준오차도 작아진다.
  • SE는 (잔차화된) 처치의 분산에 반비례. 처치 변동이 클수록 측정이 쉬워짐.
연속형 처치 실험의 표준오차 근사:
SE ≈ σ(y) / [ σ(T) · √(n − 2) ]
  • σ(T)를 키우면 SE가 줄어든다 → 실험 설계 시 처치의 분포를 신중히 선택해야.

  • 그림 4-1: 직교화로 편향 제거하기
  • 1단계: 처치 T와 결과 Y를 색(교란 요인 X)으로 표시 → 처음에는 음의 기울기.

직교화로 잡음 제거하기 (그림 4-2)
  • E[Y|X]를 추정하고 잡음 제거: Y − E[Y|X].
  • X가 Y의 분산을 많이 설명하면, 잡음 제거된 결과에서는 분산이 크게 줄어듦.
  • 마지막으로 편향+잡음 모두 제거된 T̃와 Ỹ를 그리면:
  • 깔끔한 양의 선형 관계가 명확하게 드러난다.
  • 이 마지막 회귀식은 Y를 T와 X에 동시에 회귀했을 때의 기울기와 정확히 같다.

4.4결과 모델로서의 회귀분석
  • 회귀분석은 처치를 직교화하는 방식뿐 아니라, 잠재적 결과를 대체(imputation)하는 방법으로도 볼 수 있다.
  • 대조군(T=0)에서 X에 대한 Y의 회귀분석이 E[Y₀|X]를 잘 근사한다면:
ATT = (1/N₁) ∑ 1(T_i=1) · (Y_i − μ̂₀(X_i))
  • 두 접근법을 병행하면 ATE도 추정 가능:
ATE = (1/N) ∑ (μ̂₁(X_i) − μ̂₀(X_i))
  • 회귀분석의 이중 강건성(doubly robust): E[T|X] 또는 E[Y_t|X] 중 하나만 정확해도 올바른 추정 가능.

4.5양수성과 외삽
  • 회귀분석은 모수적(parametric) 모델이므로 데이터 범위 밖으로 외삽(extrapolation)할 수 있다.
  • 중첩(overlap)이 적은 데이터셋에서도 처치효과를 추정할 수 있지만, 이는 위험할 수 있다.
  • 양수성 가정positivity: 모든 X값에서 T를 받을 확률이 0 < P(T=1|X) < 1.
  • 양수성이 충족되지 않는 영역에서는 회귀가 외삽으로 대체하는데, 이 결과를 신뢰하기 어렵다.
  • 요약: 양수성 가정이 충족되지 않으면 회귀분석은 외삽을 통해 해당 영역을 대체하게 되며, 이에 따른 위험을 감수해야 한다.
